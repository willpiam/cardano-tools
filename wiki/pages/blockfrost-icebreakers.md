# Blockfrost Icebreakers

## What it is
Icebreakers is described by Blockfrost as an incentivized quality-testing program for SPOs and other node operators running Blockfrost Platform instances to decentralize infrastructure.

## Participation signals from source
- Intended for operators willing to run platform instances as part of the decentralized fleet.
- Requires a reward address with an Icebreaker NFT License.
- Requires an Icebreaker account secret.

## Operating modes context
- **Fleet mode**: Join the Blockfrost cluster (Icebreakers context).
- **Solitary mode**: Run independently with `--solitary`, without joining the cluster.

## Dependencies and adjacent components
- A running Cardano mainnet node is a prerequisite.
- A Dolos data node is required to serve blockchain query endpoints.

## Open questions
- Source material does not specify reward mechanics or payout cadence.
- Source material does not detail acceptance criteria for joining Icebreakers.
