## Summary
Release **6.2.0** adds **treasury donation** support in the transaction builder via two new functions:

| API (as named in release notes) | Purpose |
|-----------------------------------|---------|
| `set_donation` | Donate **ADA** to the **treasury** |
| `set_current_treasury_value` | Supply the **current treasury value** when **Plutus scripts** must reference it |
